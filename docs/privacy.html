Privacy Policy

Effective Date: December 1, 2025

Time Machine for AllTrails ("the Extension") is a client-side tool designed to filter and organize photos.

Data Collection

The Extension does not collect, transmit, or store any user data on external servers. All processing happens locally within your browser's memory.

Permissions

  • ActiveTab: Used to read the AllTrails photo gallery structure to organize images.
  • Storage: Used to save your preference for "Sort Order" locally.

Contact

For questions, please open an issue on our GitHub Repository.
